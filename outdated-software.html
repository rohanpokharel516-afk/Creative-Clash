Outdated Software

The danger of running old programs

Home Digital Rights Online Safety Cyber Threats

What is it?

Programs and operating systems that haven’t been updated can contain security vulnerabilities that attackers exploit to gain access or execute malicious code.

How to Prevent It

  • Install updates and security patches as soon as they’re released.
  • Enable automatic updates where available.
  • Remove software you no longer use.
  • Consider upgrading older hardware if it no longer receives updates.
